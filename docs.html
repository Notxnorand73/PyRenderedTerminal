This page explains the PRT (PyRenderedTerminal-like) API in Python. Learn how to create terminal games and animations!

Classes

Scene(width, height, bg)

  • scene.clear() – Clears the scene to the background
  • scene.draw(x, y, char) – Draws a single character
  • scene.render() – Returns scene as string

Actor(x, y, spritesheet)

  • actor.asset(name) – Select a sprite
  • actor.move(dx, dy) – Move by delta
  • actor.goto(x, y) – Teleport to position
  • actor.stamp(scene) – Draw actor on scene

Functions

  • stamp(scene, x, y, text, transparentChar) – Stamp multi-line text
  • rect(scene, x, y, width, height, char) – Draw a rectangle
